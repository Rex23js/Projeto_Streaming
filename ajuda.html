Ajuda e Acessibilidade

Recursos de Acessibilidade

Nosso objetivo é fornecer uma experiência acessível para todos os usuários.

  • Navegação por Teclado: Todos os elementos interativos podem ser acessados usando a tecla Tab.
  • Contraste de Cores: A paleta de cores foi escolhida para garantir boa legibilidade.
  • Textos Alternativos: Todas as imagens importantes possuem descrições para leitores de tela.
